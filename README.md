Capital Sense is an advanced stock market monitoring and analysis platform tailored for Gen Z investors. 
This application provides real-time stock data, sentiment analysis, and AI-driven predictions to help users make informed investment decisions.

> **Note:** This project is currently in the **BETA** phase. Features are subject to change, and active development is ongoing.

## Features (Current)
- **Real-time Stock Data**: Live monitoring of Indonesian stocks (IHSG).
- **AI Prediction (3 Pillars)**: Advanced analysis combining Technical (Trend), Fundamental (Valuation/Profitability), and Sentiment (News) data.
- **Smart Market Recap**: Daily AI-generated market summary.
- **Restricted Access**: Simple login system for private beta testing.

## Tech Stack
- **Frontend**: Next.js 14, TailwindCSS, Framer Motion
- **Backend**: FastAPI, Python, SQLite, yfinance, Google News
- **AI/ML**: Custom heuristic models & Sentiment Analysis (BERT)
